3. If no pages match, say so clearly.
4. Format results for Telegram: use markdown, keep it concise.
5. Do not make up page content — only use what is in the index summaries.

## Input structure
Search query: <query>

Index:
(index.md content)

## Output format
Return a markdown-formatted result like:

**Search results for "<query>":**

- **[[page-slug]]** — <why this page matches, one sentence>
- **[[other-slug]]** — <why this page matches, one sentence>

If nothing matches:

Keep the response under 300 characters per result entry. Do not include pages that only tangentially relate to the query.
